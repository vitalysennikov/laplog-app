# Store Assets

Этот каталог содержит маркетинговые материалы для публикации в магазинах приложений.

## Иконки приложения

### ic_launcher_512.png
- **Размер**: 512×512 пикселей
- **Назначение**: Иконка для Google Play Store
- **Требования**: 32-bit PNG с альфа-каналом

### ic_launcher_1024.png
- **Размер**: 1024×1024 пикселей
- **Назначение**: Оригинальная иконка высокого разрешения
- **Использование**: Для создания других размеров и маркетинговых материалов

## Что еще нужно для публикации в Google Play

### Обязательные графические ресурсы:

1. **Скриншоты** (минимум 2, рекомендуется 8)
   - Телефон: 1080×1920 или 16:9 соотношение
   - Форматы: PNG или JPEG

2. **Feature Graphic**
   - Размер: 1024×500 пикселей
   - Формат: PNG или JPEG
   - Используется в заголовке страницы приложения

### Рекомендуемые:
- Скриншоты для планшетов (1536×2048)
- Промо-видео на YouTube

## Примечания

Все Android иконки (mipmap-mdpi/hdpi/xhdpi/xxhdpi/xxxhdpi) автоматически генерируются из исходного файла icon_LapLog_Free.png (1024×1024) и размещаются в app/src/main/res/mipmap-*.

См. подробную инструкцию по публикации в [GOOGLE_PLAY_PUBLISHING.md](../GOOGLE_PLAY_PUBLISHING.md).
