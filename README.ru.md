# LapLog Free - Бесплатный секундомер с историей

Русская версия | [English](README.md)

Простое и функциональное приложение-секундомер для Android с сохранением истории запусков и современным Material Design 3 интерфейсом.

## Возможности

- **Базовый функционал секундомера**: старт, пауза, сброс
- **Промежуточные отметки (Lap marks)**: сохранение промежуточных времен с отображением общего времени и длительности каждого круга
- **Переключение точности**: отображение времени с миллисекундами или только секундами
- **Динамический формат**: автоматическое переключение между форматами ММ:СС и ЧЧ:ММ:СС при достижении одного часа
- **Удержание экрана**: экран остается активным во время работы секундомера
- **Темная тема**: автоматическое переключение по системным настройкам
- **Material Design 3**: современный и интуитивный интерфейс

## Технологии

- Kotlin
- Jetpack Compose
- Material Design 3
- Android Architecture Components (ViewModel, StateFlow)
- Минимальная версия Android: 7.0 (API 24)
- Целевая версия: Android 14 (API 34)

## Сборка проекта

### Локальная сборка

1. Клонируйте репозиторий:
```bash
git clone https://github.com/vitalysennikov/laplog-app.git
cd laplog-app
```

2. Соберите APK:
```bash
./gradlew assembleRelease
```

3. APK файл будет находиться в `app/build/outputs/apk/release/`

### Сборка через GitHub Actions

Проект настроен для автоматической сборки через GitHub Actions:

1. Сделайте push в ветку `main` или `master`
2. GitHub Actions автоматически соберет APK
3. Скачайте готовый APK из артефактов в разделе Actions

APK будет доступен в течение 30 дней после сборки.

## Установка

1. Скачайте APK файл из раздела Releases или из артефактов GitHub Actions
2. Разрешите установку приложений из неизвестных источников на вашем устройстве
3. Откройте APK файл и следуйте инструкциям установки

## Использование

1. **Запуск секундомера**: нажмите кнопку "Start"
2. **Пауза**: нажмите кнопку "Pause" во время работы
3. **Продолжение**: нажмите "Resume" после паузы
4. **Сброс**: нажмите кнопку "Reset"
5. **Добавление круга**: нажмите кнопку "Lap" во время работы секундомера
6. **Переключение точности**: используйте переключатель "Show milliseconds" в верхней части экрана

## Переключатели и настройки

Приложение предоставляет несколько переключателей для настройки поведения секундомера:

### Режим экрана (`keep_screen_on`)
Циклическое переключение между тремя режимами по нажатию:
- **ВЫКЛ** (`toggle_screen_on_off`) - Экран гаснет как обычно
  - Иконка: Контурный смартфон
- **ВКЛ ПРИ РАБОТЕ** (`toggle_screen_on_running`) - Экран не гаснет только когда секундомер работает
  - Иконка: Заполненный смартфон
- **ВСЕГДА ВКЛ** (`toggle_screen_on_always`) - Экран не гаснет пока приложение открыто, даже в фоне
  - Иконка: Телефон с замком

### Блокировка ориентации (`lock_orientation`)
- **ВКЛ** - Блокирует ориентацию экрана (портрет/ландшафт)
  - Иконка: Заблокированный поворот
- **ВЫКЛ** - Разрешает поворот экрана
  - Иконка: Разблокированный поворот

### Показывать миллисекунды (`show_milliseconds`)
- **ВКЛ** - Показывает миллисекунды на основном таймере при паузе
  - Иконка: Заполненный таймер
- **ВЫКЛ** - Скрывает миллисекунды
  - Иконка: Контурный таймер

### Инвертировать цвета кругов (`toggle_invert_colors_desc`)
- **ВКЛ** - Инвертирует цвета сравнения кругов
  - Быстрые круги (-N) показываются красным
  - Медленные круги (+N) показываются зелёным
  - Иконка: Заполненные стрелки обмена
- **ВЫКЛ** - Обычные цвета
  - Быстрые круги (-N) показываются зелёным
  - Медленные круги (+N) показываются красным
  - Иконка: Контурные стрелки обмена

### Затухание яркости (`toggle_dim_brightness_desc`)
Применяется когда экран удерживается включённым (режимы RUNNING/ALWAYS):
- **ВКЛ** - Сохраняет системную яркость (фиксированно 10%)
  - Иконка: Заполненная яркость
- **ВЫКЛ** - Разрешает экрану естественно затухать после таймаута, но экран не выключится
  - Иконка: Контурная яркость

### Показывать миллисекунды в истории
Доступно во вкладке История:
- **ВКЛ** - Показывает миллисекунды в сохранённых сессиях
  - Иконка: Заполненные часы
- **ВЫКЛ** - Показывает только секунды
  - Иконка: Контурные часы
