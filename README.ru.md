# LapLog Free — Секундомер с историей кругов

Русская версия | [English](README.md)

Приложение-секундомер для Android с отметками кругов, историей сессий, графиками, резервными копиями и гибкими настройками отображения.

**Текущая версия: 0.19.0** · Минимальный Android: 7.0 (API 24) · Целевой: Android 14 (API 35)

---

## Возможности

### Секундомер
- Старт, пауза, продолжение, сброс
- Отметки кругов: два режима — только круг (пустой флажок) или круг + пауза одновременно (закрашенный флажок)
- **Основной таймер показывает время текущего круга**; общее прошедшее время — ниже в формате ММ:СС (всегда)
- Статистика кругов: среднее и середина отображаются на главном экране
- Сравнение кругов: каждый круг подсвечивается зелёным (быстрее) или красным (медленнее) относительно среднего
- Цифровой шрифт DSEG7 для основного таймера
- Миллисекунды отображаются на таймере в состоянии паузы

### История
- Все сессии сохраняются автоматически при сбросе (остановке)
- Карточки сессий с раскрытием полного списка кругов
- Наименование сессии (необязательно) с автодополнением из предыдущих сессий
- Поле заметки (необязательно, скрыто по умолчанию — открывается кнопкой-иконкой)
- Фильтр сессий по наименованию
- Табличное и карточное отображение
- Статистика сессии: среднее и середина круга

### Управление сессиями (CRUD)
- **Создание** сессий вручную через кнопку +
- **Редактирование** любой сессии: наименование, заметка, дата, время, длительность, отдельные круги
- **Удаление** сессии, удаление всех сессий до выбранной, удаление всех сессий
- Диалог редактирования с выбором даты, выбором времени (24ч) и редактированием каждого круга

### Графики
- График времён кругов по наименованию сессии
- Линии среднего и середины на графике
- Фильтр периода: Всё время / Последние 7 дней / Последние 30 дней
- Работает с любым количеством кругов (даже с одним)

### Статистика
- По каждому наименованию: количество сессий, средняя длительность, средний круг, середина круга
- Общие итоги по всем наименованиям

### Резервные копии
- Автоматическое ежедневное резервирование в выбранную пользователем папку (совместимо с Dropbox, Google Drive, MEGA)
- Ручное «Сохранить резервную копию в…» — сохранение в любое место, включая облачное хранилище
- Восстановление: заменить все данные или объединить с существующими
- Настраиваемый срок хранения резервных копий (дней)
- Удаление отдельной копии, удаление копий до выбранной, удаление всех
- Наборы переключателей по наименованиям включаются в резервную копию

### Экспорт
- Экспорт истории сессий в **CSV**
- Экспорт истории сессий в **JSON**
- Просмотр/экспорт JSON отдельной сессии

### Настройки отображения (секундомер)

| Переключатель | Функция |
|---|---|
| 📱 / 📵 Режим экрана | ВЫКЛ / ВКЛ при работе / ВСЕГДА ВКЛ (переключение по нажатию) |
| 🔒 / 🔓 Блокировка ориентации | Зафиксировать ориентацию экрана |
| ⏱ / 🕐 Миллисекунды | Показывать/скрывать миллисекунды на таймере (видны при паузе) |
| 🔄 / ↕️ Инверсия цветов кругов | Поменять местами зелёный/красный для быстрых/медленных кругов |
| ☀️ / 🌑 Сохранять яркость | При удержании экрана включённым: сохранять яркость системы или снижать до 10% после таймаута (долгое нажатие: настройка таймаута 5–300 с) |
| 🙈 / 👁 Скрыть время | Показывать мигающий заполнитель `--:--` вместо реального времени (приватность/концентрация) |
| 🔢 / ⏰ Показывать секунды | Отображать время в виде общего количества секунд вместо ММ:СС |
| 🔔 / 🔕 Тиканье | Звуковые отметки во время работы — нажатие для вкл/выкл, долгое нажатие для настройки |

### Наборы переключателей по наименованию
Каждое наименование сессии запоминает свой набор переключателей. При выборе наименования из истории его настройки восстанавливаются автоматически.

### Тиканье
- Включение/выключение звуковых отметок во время работы секундомера
- 18 типов звука: Клик, Тонк, Звонок, Бас, Высокий, Дерево, Бип, Пинг, Мягкий, Щелчок, Писк, Барабан, Перезвон, Гудок, Перезвон 2, Гонг, Чаша, Свисток
- Настраиваемые акценты: у каждого свой интервал, тип звука и начальное смещение
- **Панель предпросмотра**: воспроизведение любого звука прямо в диалоге настроек без его назначения
- По умолчанию: Перезвон каждые 60 секунд

### Работа в фоне
- Foreground-сервис удерживает секундомер запущенным при сворачивании приложения
- Постоянное уведомление показывает **название текущей активности** (например, «Бег запущен») и прошедшее время
- Язык уведомления соответствует выбранному языку приложения
- Время не отстаёт при свёрнутом приложении или выключенном экране

### Языки
Английский, русский, китайский (упрощённый). Язык определяется автоматически по системным настройкам; ручное переключение доступно в диалоге «О программе».

## Сборка

### Через GitHub Actions (рекомендуется)

Каждый push в ветку `main` автоматически собирает debug-APK:

1. Сделайте push в `main`
2. Откройте вкладку **Actions** на GitHub
3. Скачайте APK из **Artifacts** (доступен 30 дней)

### Локальная сборка

```bash
git clone https://github.com/vitalysennikov/laplog-app.git
cd laplog-app
./gradlew assembleDebug
# APK: app/build/outputs/apk/debug/
```

---

## Установка

1. Скачайте APK из Releases или артефактов Actions
2. Разрешите установку из неизвестных источников на устройстве
3. Откройте APK и следуйте инструкциям установщика

## Технологии

- **Язык**: Kotlin
- **UI**: Jetpack Compose + Material Design 3
- **База данных**: Room 2.6.1
- **Состояние**: StateFlow + ViewModel (MVVM)
- **Фон**: Foreground Service + WorkManager (плановые резервные копии)
- **Хранилище**: Storage Access Framework (SAF)
- **Графики**: Vico 2.1.3
